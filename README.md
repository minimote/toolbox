## 介绍

&emsp;&emsp;主要为安卓手表开发的 APP，**最低支持安卓 8**，主要针对方形表盘，手机也可使用。

&emsp;&emsp;工具箱，顾名思义，主要功能就是放置工具，可以把一些软件或者活动放进来，方便快速打开，例如支付宝收款码等。后续会添加一些内置小工具，如木鱼，基于 WebView 的浏览器等。
